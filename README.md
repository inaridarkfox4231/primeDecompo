# primeDecompo
素因数分解の弾幕による表現。  
数に応じてradialのシークエンスで弾丸が分裂していく感じ。  
自由に数字を決められるようにしたい(2～999)。  
[こちら](https://inaridarkfox4231.github.io/primeDecompo/) から。  
AUTO: 1ずつ増えていく。999の次は2に戻る。  
RANDOM: 2～999の勝手な値にジャンプする。  
MANUAL: 表示されているパターンが繰り返される。スライダーで自由に数を決めることができる。
